# Titanic Survival Prediction 🚢
### SoftGrowTech Data Science Internship - Task 2[Project 1]

## 📌 Project Overview
This project involves building a machine learning model to predict the survival of passengers aboard the Titanic. By analyzing factors such as age, gender, and socio-economic class, we identify the key drivers that influenced survival rates during the disaster.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
- **Algorithm:** Random Forest Classifier

## 📊 Key Insights
- **Gender:** Female passengers had a significantly higher survival rate than males.
- **Class:** Passengers in 1st Class were prioritized during rescue operations.
- **Model Accuracy:** The final Random Forest model achieved an accuracy of ~80%+.

## 🚀 How to Run
1. Clone this repository.
2. Open the `Titanic_Analysis.ipynb` in Google Colab or Jupyter Notebook.
3. Ensure `titanic.csv` is in the same directory.
4. Run all cells to see the results.
